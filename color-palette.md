# Astemari Branding Color Palette

This palette is built around Astemari’s primary identity color: **#313b28**.  
The theme reflects stability, discipline, nature, education, and quiet confidence.

---

## 1. Primary Color

### Deep Olive Green
- Hex: **#313B28**
- Usage:
  - Main brand color
  - Navigation bars
  - Primary buttons
  - Logo base
  - Headings

This is Astemari’s identity color. It should appear on every major screen.

---

## 2. Secondary Colors

### Soft Sage
- Hex: **#8FA68E**
- Usage:
  - Section backgrounds
  - Secondary buttons
  - Hover states
  - Highlights

---

### Warm Sand
- Hex: **#E6E1D6**
- Usage:
  - Main page backgrounds
  - Cards
  - Containers
  - Light surfaces

---

## 3. Accent Colors

### Muted Gold
- Hex: **#C8A951**
- Usage:
  - Call-to-action buttons
  - Icons

Used sparingly for emphasis and energy.

---

### Clay Brown
- Hex: **#7A5C3E**

### Warning Amber

### Error Red
- Hex: **#C44536**
- Usage:

---

## 6. Accessibility & Contrast Rules

- Text on **#313B28** should be **#F7F7F7**
- Text on **#E6E1D6** should be **#1E1E1E**
- Accent gold should never be used for long text
- Minimum contrast ratio: **4.5:1**

---

## 7. Brand Personality (What These Colors Communicate)

- Trust
- Calm authority
- Academic seriousness
- Natural intelligence
- Quiet confidence

No neon. No chaos. No childish gradients.
