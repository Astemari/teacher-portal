# Contributing to Teacher Portal

Thank you for your interest in contributing! Please follow these guidelines to help us maintain a collaborative and high-quality open source project.

## How to Contribute
- Fork the repository and create your branch from `main`.
- Make your changes with clear, descriptive commit messages.
- Test your changes before submitting a pull request.
- Open a pull request and describe your changes and motivation.
- Participate in code reviews and discussions.

## Code Style
- Follow Java best practices and keep code readable.
- Use meaningful variable and method names.
- Document public classes and methods with Javadoc.

## Issues
- Use GitHub Issues to report bugs or request features.
- Provide clear steps to reproduce bugs.

## Community
- Be respectful and constructive in all interactions.
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md).
